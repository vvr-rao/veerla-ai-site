# assets/

The site currently uses **inline SVG icons** (GitHub, Medium, calendar, graph),
so it is fully self-contained and needs no image files to look correct.

If you'd rather use your own PNG icons, drop them in here:

- `medium.png`
- `github.png`
- `ontology.png`
- `calendar.png`

…then swap the relevant inline `<svg>…</svg>` in `index.html` /
`ontologies/index.html` for `<img src="/assets/medium.png" alt="Medium" />`.
