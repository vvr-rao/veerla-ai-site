Ontologies

OWL ontologies I've authored for GraphRAG and document-intelligence systems. Each is class-based so document-derived mentions can be instantiated directly as members of the modeled classes. Download the .owl files below.

Veerla Intelligence Artifact Ontology

VIAO · version 0.1.0 · OWL / RDF

A lightweight ontology for representing source documents, text chunks, and generated intelligence artifacts in GraphRAG, document-intelligence, and knowledge-management systems. It distinguishes raw information sources from derived artifacts such as summaries, assertions, facts, claims, observations, insights, findings, conclusions, risks, and recommendations.

https://veerla-ramrao.ai/ontology/intelligence-artifact#

Veerla Geography Ontology

version 0.1.0 · OWL / RDF · CC BY 4.0

A class-based geography ontology for GraphRAG / document parsing. Specific continents, regions, and countries are modeled as OWL classes (with ISO alpha-2 / alpha-3 codes and region metadata) so document-derived mentions can be instantiated directly as members of those classes.

https://veerla-ramrao.ai/ontology/geography#
